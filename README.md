# Personal-Finance-Tracker

An AI-powered Personal Finance Tracker built using HTML, CSS, JavaScript, Chart.js and Google Gemini API.

## Features

- Add Income and Expenses
- Transaction History
- Delete Transactions
- Pie Chart Visualization
- Category and Month Filters
- Local Storage Support
- AI Budget Advisor using Google Gemini API

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- Google Gemini 2.5 Flash API

## How to Run

1. Clone the repository
2. Open in VS Code
3. Add your Gemini API Key in `script.js`
4. Open `index.html` using Live Server

## Project Type

Mini Project / AI Powered Web Application
